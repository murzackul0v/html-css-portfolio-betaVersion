Development

"Development" (разработка) - это процесс создания, улучшения и оптимизации программного обеспечения, веб-сайтов, приложений или других технологических продуктов.

Sowtware

"Software" (программное обеспечение) - это набор инструкций и данных, которые позволяют компьютеру выполнять определенные задачи или функции.

ООП

ООП, или Объектно-Ориентированное Программирование, это методология программирования, основанная на концепции объектов.

HTML / CSS

С помощью HTML определяются элементы, такие как заголовки, параграфы, ссылки, изображения и другие элементы контента на веб-странице.

JavaScript

JS используется для создания динамических элементов на веб-страницах, взаимодействия с пользователем, отправки запросов на сервер и многих других функций.

React

React - это библиотека JavaScript для разработки пользовательских интерфейсов. Спроектированная Facebook, она позволяет создавать веб-приложения.
